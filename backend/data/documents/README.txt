# 📚 Knowledge Base Documents

Place your PDF, TXT, or MD files here.
They will be automatically loaded and indexed into ChromaDB when the server starts.

## Supported file types:
- `.pdf` — PDF documents (lectures, syllabi, etc.)
- `.txt` — Plain text files
- `.md` — Markdown files

## How it works:
1. Drop your files in this folder
2. Restart the server (or use the upload API)
3. The RAG system will search these documents when answering questions
